# A Very Simple Project

[Corp Project](https://corp-project.vercel.app/)

## Getting Started

First, run the development server:

```bash
npm install
# or
pnpm install
#or
yarn
```

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
